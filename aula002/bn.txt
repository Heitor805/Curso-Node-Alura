instalando chalk

import chalk from 'chalk';
esse é o código a ser usado no vscode para importar a biblioteca chalk para o código, é colocado no início do programa

abrir o node e digitar "npm install chalk@5.0.1 -- save exact" o chalk serve para adicionar cores ao terminal
